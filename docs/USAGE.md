# PobTools 功能使用說明

本文說明 PobTools 各功能的操作方式。第一次安裝請先看
[安裝教學 INSTALL.md](INSTALL.md)。

---

## 一、啟動器

雙擊 `pob-zh.exe` 開啟啟動器。啟動器會自動偵測放在同一層的 POB 資料夾:

- 每個偵測到的遊戲版本會列成一列,顯示 POB 版本號與**啟動**按鈕。
- 上方工具列有翻譯編輯器、過濾器編輯器、圖譜配點器的入口。
- 底部狀態列可切換**介面語言**(繁中 / 簡中 / English)與**字型**。

### 切換語言

底部語言下拉可即時切換啟動器與 POB 介面語言。翻譯資料需對應語系存在
(預設提供 `zh-rTW`)。

### 切換字型

底部字型下拉會列出 `Fonts\` 資料夾內所有 `.ttf`。想加入自己的字型,把 `.ttf`
放進 `Fonts\` 再重開啟動器即可(建議用 TrueType 靜態字型;可變字型需先 instance
成單一字重)。預設為 Noto Sans TC(OFL 授權)。

---

## 二、翻譯編輯器

用於檢視與修改繁中字典。適合修正個別錯譯或補充新詞。

- 以英文原文搜尋條目,直接編輯對應繁中譯文。
- 儲存後即寫回 `Data\<game>\<locale>\*.json`,下次啟動 POB 生效。
- 字典是純 JSON,進階使用者也可直接編輯檔案。

---

## 三、過濾器編輯器

繁中化的物品過濾器(item filter)編輯工具,採三欄式區塊(block)編輯:

- **左欄**:過濾器內所有規則區塊清單。
- **中欄**:選中區塊的卡片檢視,顯示條件(物品類別、稀有度、階級等)與樣式
  (顏色、邊框、音效、minimap icon)。物品類別與詞彙以繁中顯示,並帶對應圖示。
- **右欄**:新增條件/樣式。
- **音效管理**分頁:管理過濾器使用的提示音效。

編輯完成後輸出仍是標準 `.filter` 檔(顯示層中文化,輸出保持遊戲可讀的英文),
可直接放進遊戲的 filter 目錄使用。

---

## 四、圖譜配點器(Atlas Planner)

規劃地圖天賦樹(Atlas passive tree)。可從啟動器工具列開啟,或用
`pob-zh.exe --atlas` 直接開啟。

### 基本操作

- **平移**:滑鼠拖曳畫布;**縮放**:滾輪。
- **配點**:點擊節點配點/取消,系統會自動計算從起點到目標的最短路徑(BFS)。
- 節點名稱與統計以繁中顯示(可用 F2 或按鈕切回英文對照)。

### 側欄統計

右側面板即時聚合目前配點的加成:

- 頂部顯示已用天賦點數與進度條。
- 搜尋框可用中英關鍵字過濾統計與節點清單。
- 點擊清單中的節點會**滑行聚焦**到該節點並標示金環。

### 多方案與分享

- 可建立多個配點方案(build),用下拉切換與增刪。
- **匯出**成 json 檔,或產生**分享碼**(`PTAT1|...`);他人可用匯入檔或貼上分享碼還原。

### 新賽季自動更新

工具列的更新按鈕(或 `pob-zh.exe --atlas-update`)會檢查 GitHub 上的最新圖譜資料,
一鍵下載並熱重載新賽季的天賦樹與圖集。繁中對照若尚未跟上,新節點會先顯示英文,
下次檢查時自動補上。
